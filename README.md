# Hangman

## About

This project features a classic Hangman game developed in Python. Designed for command-line interaction, it offers an engaging and straightforward way to play Hangman. Each milestone in the project builds upon the last, showcasing different aspects of Python programming, from basic input handling to object-oriented design.

## Getting Started

### Prerequisites

- Python 3.x: Ensure you have Python 3.x installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Installation

Clone the repository to your local machine:
```bash
git clone https://github.com/your-github-username/hangman.git
cd hangman

## Usage   

To play (Follow files to guess letters):  

python milestone_2.py

python milestone_3.py 

python milestone_4.py
